# User Stories

## Authentication
- **US-001** As a new user, I want to register with my email and password so that I can create an account and start tracking habits.
- **US-002** As a returning user, I want to log in with my email and password so that I can access my habits.
- **US-003** As a user, I want to log in with Google so that I can sign in quickly without a password.
- **US-004** As a user, I want to reset my password via email so that I can regain access if I forget it.
- **US-005** As a user, I want to be redirected to the dashboard after login so that I can immediately see my habits.

## Habit Creation
- **US-006** As a user, I want to create a new habit with a name so that I can start tracking it.
- **US-007** As a user, I want to add an optional description to a habit so that I can remember why it matters to me.
- **US-008** As a user, I want to assign a category and color to a habit so that I can organize my habits visually.
- **US-009** As a user, I want to set an optional reminder time for a habit so that I get notified to complete it each day.
- **US-010** As a user, I want to see a confirmation message after creating a habit so that I know it was saved successfully.

## Daily Tracking
- **US-011** As a user, I want to see all my active habits on the dashboard so that I know what I need to do today.
- **US-012** As a user, I want to mark a habit as complete with one click so that logging progress is fast and easy.
- **US-013** As a user, I want to unmark a habit as complete if I made a mistake so that my data stays accurate.
- **US-014** As a user, I want to see a daily progress bar showing how many habits I've completed so that I feel motivated.
- **US-015** As a user, I want completed habits to be visually distinct (e.g., greyed out with a checkmark) so that I can focus on what's remaining.

## Habit Management
- **US-016** As a user, I want to edit an existing habit's name, description, or category so that I can keep it up to date.
- **US-017** As a user, I want to archive a habit I'm no longer doing so that it doesn't clutter my dashboard but I keep its history.
- **US-018** As a user, I want to permanently delete a habit so that I can remove it and its data entirely.
- **US-019** As a user, I want to receive a confirmation prompt before deleting a habit so that I don't accidentally lose data.

## Streaks & Progress
- **US-020** As a user, I want to see my current streak for each habit so that I feel motivated to keep it going.
- **US-021** As a user, I want to see my longest streak for each habit so that I can track my personal best.
- **US-022** As a user, I want to view a calendar heatmap of a habit's completion history so that I can spot patterns.
- **US-023** As a user, I want to view my overall weekly and monthly completion rates on the Progress page so that I can evaluate my consistency.
- **US-024** As a user, I want to see total completions across all habits so that I can appreciate my cumulative effort.

## Notifications
- **US-025** As a user, I want to opt in to browser push notifications so that I get reminded to complete my habits.
- **US-026** As a user, I want to set a specific reminder time per habit so that notifications arrive when relevant.
- **US-027** As a user, I want to disable reminders for a habit without deleting it so that I have flexible control.

## Settings
- **US-028** As a user, I want to update my display name and avatar so that my profile reflects my identity.
- **US-029** As a user, I want to change my password from the settings page so that I can keep my account secure.
- **US-030** As a user, I want to toggle between light and dark mode so that I can use the app comfortably in any environment.
- **US-031** As a user, I want to delete my account and all my data so that I can leave the platform with full data privacy.
