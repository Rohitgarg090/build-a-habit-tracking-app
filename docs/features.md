# Features

## Authentication
- Email and password registration and login
- OAuth login via Google
- JWT session management via Supabase Auth
- Protected routes for authenticated users only
- Password reset via email

## Habit Management
- Create a habit with a name, optional description, category tag, and color label
- Set habit frequency (daily by default)
- Set an optional daily reminder time
- Edit any habit's details at any time
- Soft-delete (archive) or permanently delete a habit

## Daily Tracking
- View all active habits for the current day on the Dashboard
- Toggle a habit as complete or incomplete with a single click
- Completion state resets automatically each new day
- Visual indicator (checkmark, strikethrough) for completed habits
- Daily completion percentage shown as a progress bar

## Streaks & Progress
- Automatic streak calculation: consecutive days a habit was completed
- Display current streak and longest streak per habit
- Calendar heatmap on the Habit Detail screen showing completion history
- Weekly and monthly completion rate charts on the Progress screen
- Summary stats: total completions, best streak, average daily completion rate

## Notifications (Optional)
- Browser push notification reminders at user-set times
- Toggle reminders on or off per habit

## Settings & Profile
- Update display name and avatar
- Change password
- Toggle light/dark mode
- Delete account and all associated data
